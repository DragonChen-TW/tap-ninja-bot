
## Screenshot, OCR

**Description:**
This application allows users to capture a screenshot from a specified area of any desktop application window. After capturing the screenshot, it uses Optical Character Recognition (OCR) to detect and extract numbers from the image.

**Features:**
- Select a specific region of the screen or a window to capture.
- Save screenshots locally for record-keeping or further analysis.
- Use OCR to automatically detect and extract numbers from the captured image.
- Display the recognized numbers in a user-friendly interface.
- Optionally copy detected numbers to clipboard for easy use in other applications.
- Support for multiple languages and number formats.
- Error handling for failed OCR or invalid screenshot areas.

**Potential Use Cases:**
- Automating data entry from desktop applications.
- Extracting scores, statistics, or other numeric data from games or business tools.
- Assisting users with accessibility needs to quickly access numeric information.
